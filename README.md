🏎️ Find Car Price ?

Find Car Price AI is a professional-grade, full-stack machine learning application designed to provide high-precision market valuations for used vehicles. By integrating a Random Forest Regression engine with a futuristic Cyber-Tech UI, this system transforms raw automotive data into actionable insights through an immersive "diagnostic" experience.

🧠 The Intelligence Engine
The core of this application is a sophisticated AI "Brain" built using the Random Forest Regressor algorithm.

Model-Aware Logic: Unlike standard predictors that only look at a brand, this system is trained to understand specific model variants (e.g., distinguishing between a Maruti Swift and a Maruti 800).

Ensemble Learning: By utilizing 300 individual decision trees, the model achieves superior accuracy and stability, effectively handling the non-linear depreciation curves of different car types.

Data Processing: The system employs custom Label Encoding to translate human-readable categories (Fuel, Transmission, Owner type) into numerical data the AI can process instantly.

💻 Tech Stack & Architecture
This project demonstrates a complex multi-language integration, ensuring high performance and scalability.

Frontend: A high-end HTML5/CSS3 interface featuring a neon-themed design and a custom-built technical specifications database.

Backend: A robust Node.js and Express.js server that manages the application lifecycle and handles client requests.

Machine Learning: A Python 3 suite utilizing Scikit-Learn and Pandas for real-time inference and model training.

Bridge: The system uses the Child Process (Spawn) logic to create a seamless communication link between the JavaScript backend and the Python AI logic.

📁 Project Structure & Setup
To ensure the system runs correctly, files must be organized as follows:

Main Directory:

server.js — The backend controller.

predict.py — The inference logic.

train_model.py — The training script.

CAR DATASET.csv — The raw data source.

car_model.pkl & encoders.pkl — The saved AI model assets.

package.json & requirements.txt — Dependency managers.

Public Folder:

index.html — The cyber-themed user interface.

🚀 Execution Guide
To launch the application locally:

Install Dependencies: Run npm install to set up the Node environment.

Initialize AI: Run python Train_model.py to generate the latest model files.

Start System: Run npm start to activate the server.

Access: Open your browser to http://localhost:3000 to begin the car diagnostic scan.
